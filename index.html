💉 Vaccination Management System – PHP MySQL Project

A fully functional web-based vaccination center management system built using PHP, MySQL, and Bootstrap.

This system allows admins to manage hospital records, patient information, vaccination appointments, and COVID test data, all in one place.

Designed with a user-friendly interface, it ensures efficient handling of healthcare-related tasks in clinics or vaccination centers.

📌 Features

  • 🏥 Add, update, and deactivate hospital records
  • 🧑‍⚕️ Manage patients and appointments with ease
  • 📅 Schedule and view vaccination appointments
  • 🧪 Record COVID test results and statuses
  • 📊 Dashboard with live status and summary counts
  • 🌗 Dark/Light mode compatibility
  • 🔍 View further details and perform real-time actions
  • ✅ Clean sidebar navigation for smooth workflow

🛠️ Tech Stack

Frontend: HTML5, CSS3, Bootstrap 5

Backend: PHP (Core PHP)

Database: MySQL

🚀 Getting Started

🔧 Requirements

  • PHP 7.0+ (preferably 8.0)
  • MySQL Server
  • Apache Server (XAMPP, WAMP, or live server)
  • Modern web browser (Chrome, Edge, Firefox)
